==========================
TYPO3 Exception 1717495996
==========================

.. include:: /If-you-encounter-this-exception.rst.txt

Function value token stream has not been set
======================================================================================

- Try deleting the code cache (var/cache/code)

Possible Reason
===============================
This can be caused by having obsolete TSConfig/Typoscript AST parser cache data, e.g. due to the changes introduced by the fix for https://forge.typo3.org/issues/103671
