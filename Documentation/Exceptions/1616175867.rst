==========================
TYPO3 Exception 1616175867
==========================

.. include:: /If-you-encounter-this-exception.rst.txt


11.5.11 - 26.06.2022
====================================

Situation
---------

.. code-block:: none
   :caption: Exception message upon failed login
   
   (1/1) #1616175867 TYPO3\CMS\Core\RateLimiter\RequestRateLimitedException
   
   The login is locked until 26-06-22 12:33 due to too many failed login attempts from your IP address.

This exception is thrown, when there have been too many failed login from one IP Adress within a configured time.

See also: :docs:`t3coreapi:Changelog/11.3/Feature-93825-RateLimitingForFailedLogins.html`.

Solution
--------

1. Wait for the lock to time out.
2. If the limit it to low for your purpose, make the limit higher.
