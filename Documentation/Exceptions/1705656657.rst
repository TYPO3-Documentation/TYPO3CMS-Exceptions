==========================
TYPO3 Exception 1705656657
==========================

.. include:: /If-you-encounter-this-exception.rst.txt


TYPO3 13.0 - 30 April 2023
==================================

The Issue
---------

This exception is triggered if you attempt to view a page that lacks a TypoScript template.

Solution
--------

:ref:`Create a TypoScript template <t3sitepackage:typoscript-configuration>` for your site in the site's root page.
