TYPO3 Exception 1443793404
==========================

.. include:: /If-you-encounter-this-exception.rst.txt


TYPO3 12.4 - 01.07.2025
=======================

..  warning::

    Inline field fieldname of table tx_myextension_domain_model_mytable must have a foreign_table config

Installation Overview
---------------------

Custom extension with custom table and inline field created by :composer:`stefanfroemken/ext-kickstarter`

The Issue
---------

The generated TCA is not complete:

..  code-block:: php

    'fieldname' => [
        'label' => 'Fieldname',
        'config' => [
            'type' => 'inline',
        ],
    ],

Solution
--------

Add foreign_table and foreign_field to TCA:

..  code-block:: php

    'fieldname' => [
        'label' => 'Fieldname',
        'config' => [
            'type' => 'inline',
            'foreign_table' => 'tx_myextension_domain_model_othertable',
            'foreign_field' => 'parent',
        ],
    ],

See also `Example: Simple 1:n inline relation <https://docs.typo3.org/permalink/t3tca:tca-example-inline-1n-inline-1>`_
