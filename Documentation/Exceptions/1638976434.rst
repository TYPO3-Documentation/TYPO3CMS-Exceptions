==========================
TYPO3 Exception 1638976434
==========================

.. include:: /If-you-encounter-this-exception.rst.txt


TYPO3 13.4.27 - 22.04.2026
==========================

Installation Overview
---------------------

This error can happen if you try to use :php:`TYPO3\CMS\Extbase\Utility\LocalizationUtility` 
in the wrong context, for example within an enums label that is also called from TCA.

Solution
--------

Get the translation service from the current backend user in backend context.
